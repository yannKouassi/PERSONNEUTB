Projet Java – Gestion de personnes  
Développement d’une application Java orientée objet permettant la gestion d’un registre de personnes. Le projet intègre une hiérarchie de classes (Personne, PersonnesGenre, Humain) et utilise des structures génériques pour organiser les données par sexe et matricule. Les fonctionnalités incluent l’ajout, la recherche, le comptage, la suppression et l’affichage des personnes, avec persistance des données dans un fichier texte. Ce projet illustre la maîtrise des collections (HashMap, Iterator), de la gestion des entrées utilisateur (Scanner), de la persistance (FileWriter, BufferedWriter) et des bonnes pratiques de programmation orientée objet en Java.
